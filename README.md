# Wine Quality

Predicting whether a red wine is "good" (quality ≥ 7) from 11 chemical
measurements. Uses cross-validation, hyperparameter tuning, and compares
logistic regression, random forest, and gradient boosting on the
UCI wine quality dataset.

## Setup

```bash
uv venv --python 3.12
source .venv/bin/activate
uv pip install pandas matplotlib seaborn scikit-learn jupyter
```

Open `wine.ipynb`.
